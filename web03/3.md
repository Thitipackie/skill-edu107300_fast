# 新增api.php

規劃switch，並開始做login的提交作業

```php
<?php
include "sql.php";
switch ($_GET['do']) {
  case 'login':
    if ($_POST['acc'] == "admin" && $_POST['pwd'] == "1234") {
      $_SESSION['admin'] = 123;
      plo("admin.php");
    } else echo "<scrip>alert('輸入錯誤');window.history.back()</scrip>";
    break;
}
?>
```

另外這裡不需要設計登出功能，因為前後台都能同時操作。
