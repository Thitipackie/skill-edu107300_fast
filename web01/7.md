# 增添sql.php

### 2. 前台的total顯示格式

增加以下代碼做提取。同時判別如果沒SESSION紀錄，先update值+1並更新此值。

```php
//t7
$re=select("t7_total",1);
$t7num=$re[0]['once'];
if(empty($_SESSION['visit'])){
    $_SESSION['visit']=123;
    $_POST['once']=$t7num+1;
    update($_POST,"t7_total");
    $t7num++;
}
```
